# Juego Java 2D 

Motor de juego 2D hecho desde cero usando Java y AWT

## Características
- Ventana con Canvas
- Game loop propio
- Control de FPS
- Render con BufferStrategy

## Cómo ejecutar
1. Importar proyecto en Eclipse
2. Ejecutar clase Juego

## Estado
En desarrollo 

## Objetivo
Proyecto para aprender desarrollo de videojuegos
